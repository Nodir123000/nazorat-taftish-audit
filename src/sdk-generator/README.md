# 🏭 SDK Generator

Мы не пишем HTTP-клиенты вручную. Мы генерируем их из спецификации OpenAPI.

## 🛠 Инструменты

Мы используем `swagger-typescript-api` для генерации типизированного `Axios` клиента.

## 🚀 Как использовать

1. Убедитесь, что спецификация актуальна: `docs/openapi.yaml`.
2. Запустите генератор:

    ```bash
    npm run generate:sdk
    ```

3. Сгенерированный файл появится в: `lib/api/generatedless-sdk.ts`.

## ⚙️ Конфигурация

Скрипт находится в `scripts/generate-sdk.sh`.
Он:

1. Валидирует YAML.
2. Генерирует TypeScript типы (DTO).
3. Генерирует API класс для вызова эндпоинтов.

## ⚠️ Правила

* ⛔️ **НЕ РЕДАКТИРУЙТЕ** файл `lib/api/generated-sdk.ts` вручную. Ваши изменения затрутся.
* ✅ Если нужен новый метод — добавьте его в OpenAPI.
* ✅ Если нужно изменить поведение (например, Auth headers) — используйте инстанс `axios` в `lib/api/http-client.ts`.
