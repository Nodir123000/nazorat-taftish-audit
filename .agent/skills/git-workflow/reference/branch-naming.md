# Branch Naming Quick Reference

| Task | Branch name |
|------|-------------|
| Add new violation type | `feature/PS-04-add-corruption-risk-type` |
| Fix deadline calculation | `fix/PS-05-deadline-workday-calculation` |
| Add Excel report | `feature/PS-06-violations-registry-excel` |
| PINFL security fix | `fix/PS-07-mask-pinfl-inspector-role` |
| Refactor audit service | `refactor/PS-03-audit-service-cleanup` |
| Update Prisma schema | `chore/PS-00-add-created-at-to-documents` |
| Add E2E tests | `test/PS-03-audit-lifecycle-e2e` |
| Update ТЗ docs | `docs/PS-00-tz-update-section-3` |
| Fix auth bug | `fix/PS-01-session-expiry-not-redirecting` |
| Add unit tests | `test/PS-04-violations-service-unit` |
| Add new ПС module | `feature/PS-11-notifications-module` |

**Never use:** `main`, `master`, `dev`, `test`, `temp`, `wip`, `fix-bug`, `new-feature`, `update`
