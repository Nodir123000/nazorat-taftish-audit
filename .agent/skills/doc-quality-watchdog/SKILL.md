# Doc-Quality-Watchdog Skill

Этот скилл предназначен для автоматического поддержания качества документации проекта в формате Markdown.

## Основные возможности

1. **Авто-исправление иерархии (MD025, MD041)**:
   - Гарантирует, что файл начинается с заголовка `#`.
   - Понижает дублирующиеся заголовки `#` до `##`.
2. **Форматирование таблиц (MD060)**:
   - Автоматически добавляет пробелы вокруг разделителей (`| --- |`).
   - Сохраняет выравнивание колонок.
3. **Управление пространством (MD022, MD031)**:
   - Добавляет пустые строки вокруг всех заголовков и списков.
   - Убирает лишние пустые строки.
4. **Метаданные кода (MD040)**:
   - Добавляет `text` ко всем немаркированным блокам кода ` ``` `.

## Использование

Для запуска автоматического исправления всей документации в папке `docs/ТЗ`:

```bash
npm run docs:fix
```

Или напрямую через tsx:

```bash
npx tsx .agent/skills/doc-quality-watchdog/scripts/remediate.ts
```

## Интеграция

Скилл интегрирован в `package.json` и может быть запущен перед фиксацией изменений (pre-commit) или по запросу пользователя.
