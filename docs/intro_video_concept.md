# Креативный бриф и концепция интро-видео
**Проект:** Интро для АИС «Taftish ma’lumotlar bazasi»
**Заказчик:** Управление контроля и ревизии Министерства обороны (Mudofaa vazirligining Nazorat-taftish boshqarmasi)

---

## 1. Назначение и статус
Система представляет собой закрытый цифровой контур государственного значения, предназначенный для финансового контроля, аудита и аналитики. Интро-видео служит не для развлечения, а для **цифровой идентификации ведомства** и настройки пользователя на рабочий, ответственный лад.
**Визуальный язык:** Институциональный, строгий, "Digital Military". Отражает непроницаемость, точность и порядок.

## 2. Основное сообщение (Key Message)
"Системность. Безопасность. Государственный контроль."
Пользователь за 2 секунды считывает: *Я нахожусь в официальном цифровом пространстве Министерства обороны. Здесь действуют строгие регламенты.*

## 3. Арт-дирекшн (Визуальный стиль)
*   **Цветовая палитра:**
    *   *Фон:* Глубокий, сложный темный цвет (Navy Blue / Deep Slate). Например: `#0B1120` (основа) с легким градиентом в `#1E293B` (верх) для объема.
    *   *Акценты:* Благородное золото (для элементов герба) и холодный белый/стальной (для текста).
*   **Графика:**
    *   Минимализм. Отсутствие декоративных партиклов, вспышек и "игровых" эффектов.
    *   Геометрия: Тонкие линии, сетки, символизирующие базы данных и структуру.
    *   Анимация: Плавная (Eased), уверенная, без рывков. Движение объектов "тяжелое", "весомое".

## 4. Посекундный сценарий (Хронометраж: 6-7 секунд)

| Время | Действие (Action) | Визуальное описание (Visuals) |
| :--- | :--- | :--- |
| **00:00 - 00:02** | **Вход в контур** | Экран черный. Плавно (Fade In) проявляется глубокий темный фон. На заднем плане с прозрачностью 5-10% прорисовывается контурная карта Республики Узбекистан или абстрактная сетка защищенного соединения. Камера делает очень медленный "наезд" (Zoom In). |
| **00:02 - 00:04** | **Иерархия** | В верхней части экрана проявляется надпись (шрифт строгий sans-serif, uppercase):\
`MUDOFAA VAZIRLIGINING`\
`NAZORAT-TAFTISH BOSHQARMASI`\
Анимация появления: буквы не вылетают, они плавно проявляются через прозрачность (Opacity) и легкий сдвиг снизу-вверх (Y-axis shift). |
| **00:04 - 00:06** | **Центральный объект** | По центру экрана, **массивно и четко**, появляется название системы:\
**«TAFTISH MA’LUMOTLAR BAZASI»**\
Ниже, меньшим кеглем: *avtomatlashtirilgan axborot tizimi* \
Одновременно над текстом проявляется **Герб Министерства обороны**. Он не вращается. Он появляется через эффект "расфокус -> фокус" (Blur Reveal), подчеркивая ясность и прозрачность контроля. |
| **00:06 - 00:07** | **Фиксация (Lockup)** | Вся композиция (Герб + Название ведомства + Название системы) замирает. Легкий блик света (Sheen effect) проходит по канту герба или золотым буквам, подтверждая подлинность. |
| **00:07+** | **Переход** | Плавное растворение (Cross Dissolve) в интерфейс формы входа. Видео не должно резко обрываться черным экраном. |

## 5. Финальный экран (Компоновка)
Строгая центровая композиция (Center Aligned):

1.  **(Верх)** Герб Министерства Обороны (полноцветный, но с коррекцией под темный фон) — размер ~15-20% высоты экрана.
2.  **(Центр)** Название ведомства (Small Caps, Spaced) — цвет "Steel Grey".
3.  **(Центр, Крупно)** **TAFTISH MA’LUMOTLAR BAZASI** — цвет "White", шрифт жирный (Bold/Semibold).
4.  **(Низ)** *avtomatlashtirilgan axborot tizimi* — цвет "Muted Blue/Grey".

## 6. Технические рекомендации к реализации
*   **Формат:** `WebM` (приоритет для веб) и `MP4` (H.264) как фоллбек.
*   **Разрешение:** Оптимально `1920x1080` (Full HD). Для 4K мониторов векторные элементы интерфейса системы важнее видео.
*   **Частота кадров:** `60 fps`. Плавность критически важна для ощущения "дорогого" продукта.
*   **Звук:**
    *   *Рекомендация:* Полное отсутствие музыки.
    *   *Опционально:* Едва слышный, низкий "атмосферный гул" (Drone Sound) и тихий звук "затвора" или "печати" в финале. Никакой эпичной оркестровой музыки.
*   **UX:**
    *   Видео должно проигрываться **один раз** при загрузке страницы.
    *   Если пользователь обновил страницу (F5), видео лучше не показывать повторно (сохранять флаг в SessionStorage), сразу показывая форму входа.
    *   Обязательно предусмотреть возможность клика мышкой по любой области для мгновенного пропуска (Skip).
