## Е.1. ОБЩИЕ ПОЛОЖЕНИЯ

### Е.1.1. Назначение

Настоящее приложение описывает основные бизнес-процессы АИС КРУ МО РУЗ в нотации BPMN 2.0. Документ включает текстовые описания процессов в табличной форме, достаточные для понимания логики без визуальной диаграммы.

### Е.1.2. Условные обозначения BPMN (табличная форма)

| Элемент | Обозначение | Описание |
|---------|------------|---------|
| **Start Event** | ○ | Начало процесса |
| **End Event** | ◉ | Конец процесса |
| **Task** | □ | Задача (выполняется участником) |
| **User Task** | ⬜ | Задача с участием пользователя |
| **Service Task** | ▣ | Автоматическая задача |
| **Exclusive Gateway** | ◇ | Развилка «ИЛИ» (только один путь) |
| **Parallel Gateway** | ⊕ | Развилка «И» (все пути параллельно) |
| **Sequence Flow** | → | Переход |
| **Pool** | ═ | Исполнитель (Pool) |
| **Lane** | │ | Роль/Подразделение (Lane) |
