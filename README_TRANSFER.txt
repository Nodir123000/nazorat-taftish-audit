ИНСТРУКЦИЯ ПО ПЕРЕНОСУ ПРОЕКТА
================================

1. Скопируйте всю папку с проектом на флешку (кроме папки node_modules, чтобы сэкономить место).
2. Убедитесь, что в корне лежат файлы:
   - db_transfer_ready_2026_04_27.sql (Дамп базы)
   - setup-on-new-pc.ps1 (Этот скрипт)
3. На НОВОМ компьютере:
   - Установите Node.js (https://nodejs.org)
   - Установите PostgreSQL (https://www.postgresql.org)
4. Откройте папку с проектом на новом ПК.
5. Нажмите правой кнопкой мыши на файл `setup-on-new-pc.ps1` и выберите "Выполнить с помощью PowerShell" (Run with PowerShell).
6. Следуйте инструкциям в окне терминала (скрипт попросит ввести пароль от вашей новой базы PostgreSQL).

После завершения работы скрипта, введите в терминале:
npm run dev

Система будет запущена!
