
# KRR System (Nazorat Taftish)

Система автоматизации контроля и ревизий (КРУ).

## 🚀 Как запустить проект

### Локально

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Подготовьте базу данных:

   ```bash
   npm run db:seed
   ```

   *Это создаст базу и наполнит её тестовыми данными (Справочники, Люди, ВУС).*

3. Запустите сервер:

   ```bash
   npm run dev
   ```

### ☁️ GitHub Codespaces

1. Нажмите зеленую кнопку **<> Code** -> **Codespaces** -> **Create codespace on main**.

2. В терминале Codespaces выполните:

   ```bash
   npm install
   npm run db:seed
   npm run dev
   ```

## ✅ Статус системы

- **Справочники**: Восстановлены (ВУС, Звания, Должности).
- **База данных**: 1800+ записей физических лиц.
- **Инфраструктура**: Prisma ORM + PostgreSQL (SQLite в dev-режиме).
