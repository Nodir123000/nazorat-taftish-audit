# Инструкция по развертыванию системы (Nazorat Taftish)

## 1. Подготовка окружения
Перед запуском убедитесь, что на новом компьютере установлены:
*   **Node.js** (версия 18.0 или выше). [Скачать](https://nodejs.org/)
*   **PostgreSQL** (версия 14 или выше). [Скачать](https://www.postgresql.org/download/)
*   **Git** (необязательно, если копируете файлы через флешку).

## 2. Копирование проекта
Скопируйте папку проекта на новый компьютер.
> **Важно:** Папку `node_modules` и `.next` копировать НЕ НУЖНО. Они весят много и пересоздаются автоматически.

## 3. Настройка базы данных
1.  Убедитесь, что PostgreSQL запущен.
2.  Создайте пустую базу данных (например, `nazorat_db`) через pgAdmin или консоль.

## 4. Настройка конфигурации (.env)
В корне проекта создайте файл `.env` (или скопируйте его с старого ПК). Внутри должны быть настройки подключения к БД:

```env
# Пример настройки (измените пароль и имя базы на свои)
DATABASE_URL="postgresql://postgres:ваш_пароль@localhost:5432/nazorat_db?schema=public"
```

## 5. Установка и запуск
Откройте терминал (PowerShell или CMD) в папке проекта и выполните команды по очереди:

### Шаг 1: Установка зависимостей
```bash
npm install
```

### Шаг 2: Создание структуры базы данных
Эта команда создаст все таблицы в вашей новой базе данных.
```bash
npx prisma db push
```

### Шаг 3: Генерация клиента Prisma
```bash
npx prisma generate
```

### Шаг 4: Загрузка справочников (Территорий)
Чтобы в базе появились области и районы из Excel файлов (`Областя.xlsx`, `Районы.xlsx`), выполните:
```bash
npx tsx scripts/reset-and-seed-territories.ts
```

### Шаг 5: Запуск системы
Для разработки (с возможностью правки кода):
```bash
npm run dev
```
После этого откройте в браузере: `http://localhost:3000`

Для полноценной работы (быстрее, но без режима правки):
```bash
npm run build
npm start
```
